2/**
3 * Contract that every AI provider must satisfy.
4 *
5 * Agnosis uses distinct AI operations per media type:
6 *   1. describe()    — read the image + artist prompt and produce publication-ready text.
7 *   2. enhance()     — return an improved version of the image binary.
8 *   3. transcribe()  — convert audio binary to a text transcript.
9 *   4. chat()        — lightweight text-in / text-out for classification and description tasks.
10 *
11 * Providers may implement one or more of these operations. The Pipeline checks
12 * supports_enhancement() and supports_audio() before calling enhance()/transcribe().
13 *
14 * @package Agnosis\AI
15 */
