3 * Transient-based per-IP rate limiter.
4 *
5 * Counts requests per action per IP address within a sliding time window and
6 * returns a WP_Error when the limit is exceeded.  Uses WordPress transients so
7 * it works on any host without extra infrastructure.
8 *
9 * Usage in a permission callback:
10 *
11 *   $rate = RateLimiter::check( 'admission_apply', 5, 60 );

18 * (e.g. via nginx real_ip_from) work correctly without any additional config.
