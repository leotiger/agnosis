3 * Agnosis Node Identity.
4 *
5 * Each Agnosis installation is a node in the rhizome.
6 * This class generates and maintains the node's cryptographic identity
7 * (RSA key pair) and exposes a /.well-known/agnosis-node endpoint
8 * that other nodes can read to verify and federate with this one.
9 *
10 * GET /wp-json/agnosis/v1/node          — node identity card
11 * GET /.well-known/agnosis-node         — lightweight node discovery
12 * POST /wp-json/agnosis/v1/node/peers   — register a peer node
13 *
14 * register_peer() is intentionally reachable without WordPress auth (any
15 * fediverse/rhizome node, not a logged-in user, calls it) — it is instead
16 * gated by three independent controls added for audit §2d: a per-IP rate
17 * limit, a cap on total pending rows, and a requirement that the request be
18 * signed by the private key matching the public key it presents (see
19 * `HttpSignature::verify_with_key()`). A row landing here is still only
20 * ever `status = 'pending'`; `list_peers()` exposes `trusted` rows only, and
21 * nothing in this codebase promotes a row to `trusted` automatically.

173    /**
174     * Prune the oldest `status = 'pending'` rows once the table has reached
175     * MAX_PENDING_PEERS, so a new registration always has room (audit §2d).
176     * Signature verification (see register_peer()) stops an attacker from
177     * forging someone ELSE's identity, but it does nothing to stop unlimited
178     * freshly-minted keypairs registering unlimited distinct URLs — this cap
179     * is the backstop for that.

183    private function enforce_pending_peer_cap( \wpdb $wpdb ): void {

197    private function ensure_key_pair(): void {
198        if ( get_option( 'agnosis_public_key' ) && get_option( 'agnosis_private_key' ) ) {

201
202        // openssl_pkey_new() with a 2048-bit RSA key reads from /dev/random, which

210        }

214        }

216        $key = openssl_pkey_new( [ 'digest_alg' => 'sha256', 'private_key_bits' => 2048, 'private_key_type' => OPENSSL_KEYTYPE_RSA ] );
217        if ( false === $key ) {
