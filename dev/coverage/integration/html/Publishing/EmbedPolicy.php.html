24 * Three-tier trust model:
25 *
26 *   0. Community bypass (opt-in, off by default via
27 *      `agnosis_embed_trust_community`) — every artist here was already
28 *      vouched in by the community during admission. A site admin who

32 *      explicit opt-in — off by default — because it removes every other
33 *      safeguard in this class at once.
34 *
35 *   1. Trusted-host fast path — a configurable list of known platforms
36 *      (YouTube, Vimeo, SoundCloud, Bandcamp, …). No network request, no AI
37 *      call. Was previously a hardcoded PostCreator constant; now a Settings
38 *      field (`agnosis_embed_trusted_hosts`) so a site admin can add or
39 *      remove platforms without a plugin update, on top of the developer-
40 *      facing `agnosis_embed_host_allowlist` filter, which still exists for
41 *      the same purpose it always has.

43 *   2. AI review (opt-in, off by default via `agnosis_embed_ai_vetting_enabled`)
